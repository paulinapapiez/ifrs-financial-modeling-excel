📊 Financial Statement Analysis (IFRS) – Case Study
🔎 Project Overview

This project presents a comprehensive financial statement analysis based on IFRS standards.
The analysis includes:

Statement of Financial Position (Balance Sheet)

Income Statement (Profit & Loss)

Cash Flow Statement

Financial ratio analysis

Business conclusions and performance evaluation

The model was built entirely in Microsoft Excel.

🎯 Objective

The goal of this project was to:

Analyze a company’s financial condition using IFRS-based statements

Evaluate liquidity, profitability, solvency, and efficiency

Interpret financial performance from a business perspective

Draw strategic conclusions based on financial data

📂 Project Structure

The Excel model contains:

Introduction – business and analytical context

Balance Sheet (IFRS) – structured financial position

Income Statement (IFRS) – performance analysis

Cash Flow Statement – operating, investing, financing flows

Financial Ratio Analysis – key KPIs:

Liquidity ratios

Profitability ratios

Leverage ratios

Efficiency ratios

Conclusions – interpretation and strategic insights

🛠 Tools Used

Microsoft Excel

Financial modeling techniques

Ratio analysis

IFRS reporting structure

💡 Key Skills Demonstrated

Financial analysis

Data structuring

Business reasoning

Analytical thinking

Financial statement interpretation

Excel modeling

📈 Key Insights

- The company maintains stable liquidity but shows decreasing profitability.
- Cash flow analysis indicates strong operational performance.
- Debt ratio suggests moderate financial risk.
